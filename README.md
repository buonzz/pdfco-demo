# pdfco-demo

This is a demo app for using https://pdf.co/ API


## Requirements

* an account with https://pdf.co/ (with enough credits)
* NodeJS 8 or higher

### Installation

clone this repo

```
git clone git@github.com:buonzz/pdfco-demo.git
```

install the dependencies

```
npm install
```

set your api keys
```
cp env.example .env
```
